
1. ng build --prod (or ng build)
2. npm install -g firebase
3. firebase -init --debug (use "dist/miles-challenge" as the project directory setting, where thee index.html is located)
4. firebase deploy
5. firebase open hosting:site

to test locally:
firebase serve
